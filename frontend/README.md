Installation

Clone the repository:

git clone https://github.com/anjalideshmukh969/StackOracle.git
cd frontend

Install dependencies:

npm install

Start development server:

npm run dev

Application will run on:

http://localhost:5173


Build for Production
npm run build

Preview production build:

npm run preview
